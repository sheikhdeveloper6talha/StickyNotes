I am a student of Saylani Mass IT Training, and this Sticky Notes Web Application was assigned to me as a project task.
I have developed a Sticky Notes Web Application with a complete authentication system including Login and Signup functionality.
This project uses localStorage as a database to store user data and notes, allowing data persistence without a backend server.
After successful login, users are redirected to their personal dashboard where only their own created notes are displayed. Each user can manage their notes independently within their dashboard.
The Home Page displays all notes created by all users, allowing a shared viewing experience. When a user clicks on any note, a detailed page opens where all related notes of that specific user are shown.
This project also includes Private Routing for secure access control. Unauthenticated users are restricted from accessing the dashboard and personal notes pages. Only logged-in users can access protected routes after authentication.
Key Features:


User Authentication (Login / Signup)


Private Routing for secure pages


User-specific Dashboard


Global Home Page showing all users’ notes


Detailed Notes View on click


localStorage used as database for data persistence


Secure access control system


This assignment helped me strengthen my understanding of authentication, routing, and state management in modern web applications.
